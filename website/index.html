Personal AI infrastructure

An iMessage-native agent that can route work, remember context, and create focused sub-agents.

Mia is a local-first monorepo built with FastAPI, LangGraph, Convex, SendBlue, and Next.js. It receives iMessages, stores memory, runs agent workflows, and exposes a realtime control surface for debugging and setup.

View on GitHub Join Discord

System

Built as real agent infrastructure, not a chatbot wrapper.

01

iMessage interface

SendBlue webhooks receive inbound messages, dedupe events, and send replies back through iMessage.

02

Parent router

LangGraph classifies user intent and creates dynamic sub-agent specs with isolated tool access.

03

Memory system

Convex stores short-term, long-term, and permanent memories with importance and decay metadata.

04

Memory Court

A nightly workflow consolidates, challenges, and judges memory cleanup decisions with audit records.

05

Gateway

A single local process starts the dashboard, Convex dev, and the Python agent service.

06

Dashboard

A realtime Next.js interface shows agents, memory, events, automations, connectors, and setup state.

Get Started

Run Mia locally.

1. Clone the repository and install dependencies.

git clone https://github.com/zhiliao000-star/mia-agent.git
cd mia-agent
npm install

2. Set up your environment variables by copying .env.example to .env and filling in the required values.

cp .env.example .env

3. Run the Mia Gateway to start all services.

npm run mia:gateway
